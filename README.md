# uber_project
This is a simple Uber homepage clone created using HTML and CSS.
The project focuses on building a responsive and clean UI similar to the official Uber website.

🔹 Features

Responsive navigation bar

Hero section with background image

Service sections (Ride, Drive, etc.)

Footer with links

Clean and modern UI design

🛠️ Technologies Used

HTML5

CSS3

📂 Project Purpose

This project was created for practice to improve:

Web layout designing

CSS styling

Responsive design skills
